# 概要

https://wekakeibo.herokuapp.com/

支出の記録のみのシンプルな家計簿です。


支払いの入力を現金で支払っているような感覚の(実装はできていません)家計簿を作成したいと思いました。
最近は,キャッシュレスが増え,現金のような手間をかけずに支払いができるようになりました。
便利である一方で現金感覚が身に付かない子供がいるというニュースを見て作成してみようと思いました。
